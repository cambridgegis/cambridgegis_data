# Public Art
## GIS File Name
LANDMARK_PublicArt
## Description
<DIV STYLE="text-align:Left;"><DIV><DIV><P><SPAN>This point layer contains locations of over 300 catalogued public artworks located both indoor and outdoor within the City of Cambridge. There are six types of points including sculptures, murals, mosaics, stained glass, landscape design, and electronic artworks. The Cambridge Arts Council manages and maintains these artworks, while continuing to commission new works through the City’s Percent-for-Art program.</SPAN></P></DIV></DIV></DIV>

## Purpose
Created for general use within Cambridge Arts Council, for map production, and for public viewing.
