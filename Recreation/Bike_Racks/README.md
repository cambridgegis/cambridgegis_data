# Bike Racks
## GIS File Name
RECREATION_BikeRacks
## Description
<DIV STYLE="text-align:Left;"><DIV><DIV><P><SPAN><SPAN>Bicycle parking data for public land in the city has been created through three methods: site surveys, detection through aerial imagery, and installation tracking. The data does not include bicycle parking on private property and excludes some racks that do not meet the city’s standards for rack design.</SPAN></SPAN></P></DIV></DIV></DIV>

## Purpose
Bicycle parking data for public land in the city.
